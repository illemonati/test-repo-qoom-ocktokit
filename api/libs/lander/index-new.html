What is Qoom?

Create Web Apps Together

Qoom provides your club with a piece of the cloud that makes it easy to create applications without the hassle of GitHub, terminal, and other distracting processes allowing your team to just focus on turning your idea into reality

Share Your Creations

Qoom provides all its member access to a social platform to garner feedback and users required to make your product have an impact on others

Promote Yourself

Qoom provides an outlet to highlight all your clubs and individual accomplishments to make it easy to help you win competitions, get into a prestigious college or get a great start on your career

Start Coding on Qoom

Individual

Qoom is a collaborative programming environment where the platflorm lives in your own domain and is accessible through the browser. Any application you create will be instantly deployed to your web server and will be instantly accessible through your own domain name from any device that is connected the web. The diversity of projects you can create and share range from platform-independent mobile web apps to APIs that can be accessed from an Arduino controlling your personal robot. You have control of what goes into the web app from its view to API to a database.

Clubs

Collaboratively work with other members of a club to create a web application. NO need for Github, the code working on your domain name will keep track of all your changes as well as give you a chance to rollback. Clubs also will be able to create a club custom domain name allowing club members choose how code gets into production. Members will have a choice to create subdomains off of the club domain account, giving them a chance to create a near-infinite number of web applications.
